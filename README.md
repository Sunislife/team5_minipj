# 🚇 서울시 지하철 유동인구와 상권 매출의 상관관계 분석

> SCQA 기반 데이터 분석을 통한 창업 입지 추천 프로젝트

## 📌 프로젝트 소개

서울 지하철 역별 유동인구와 상권 매출 데이터를 결합하여 **"유동인구가 많은 곳이 반드시 매출이 높은 상권일까?"** 라는 질문에 데이터로 답하고자 한 프로젝트입니다.

단순히 유동인구만을 기준으로 상권을 평가하는 기존의 관점을 검증하고, 실제 소비가 발생하는 상권을 데이터 기반으로 분석하였습니다.

## SCQA

- S : 상황 설명
- C : 현재 상황이 지속되면 어떤 일이 발생하는가?
- Q : C를 해결하기 위한 질문
- A : 질문에 대한 답 - Top Message (Acton 형태로 답 나오는 것이 베스트)

## 🔍 주요 분석 결과

### 회귀분석

* R² = **0.164**
* 유동인구는 매출을 약 **16.4%만 설명**

### Spearman 상관분석

* ρ = **0.574**
* 유동인구 순위와 매출 순위는 완전히 일치하지 않음

### 집단 비교

* t-test : p < 0.001
* ANOVA : p < 0.001

→ 매출이 높은 그룹일수록 평균 유동인구는 많았지만, 동일한 유동인구에서도 매출 차이가 크게 나타남.

## 💡 핵심 인사이트

* 유동인구가 많다고 반드시 매출이 높은 것은 아니다.
* 소비 목적과 체류 시간이 상권의 성과에 큰 영향을 준다.
* 유동인구는 상권 분석의 필요조건이지만 충분조건은 아니다.
* 창업 입지는 유동인구와 실제 매출을 함께 고려해야 한다.
